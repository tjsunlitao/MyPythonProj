#### 气缸块说明
Basic Control
===
TransferBasic
---
if 不是复位过程中：
    输出 = 手自动控制点(手动带安全互锁)
elif 复位过程中：
    输出 = 复位控制点(直接控制模式带自锁)
    if 复位过程中双向控制点都TRUE:
        输出 = 1
LatchiBasic
---
if 不是在不锁存模式：
    if 输出：
        if Jog控制模式：
            输出自锁(到达原始位传感器)
        else:
            输出自锁

Work Control
===
TransferWork
---
if 不是在复位过程中：
    输出 = 手自动控制点(手动带安全互锁)
elif 复位过程中：
    输出 = 复位控制点(直接控制模式带自锁)
    if 复位过程中双向控制点都TRUE:
        输出 = 1
LatchiWork
---
if 不是在不锁存模式：
    if 输出：
        if Jog控制模式：
            输出自锁(到达工作位传感器)
        else:
            输出自锁
#### 初始化输入
===
- 复位动作完成后初始化所有输入控制点
- 不是复位动作中一直复位复位控制点
- 允许自动线圈输出时复位自动控制点
- 未缩回/未伸出/双传感器报警
